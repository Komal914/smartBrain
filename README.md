## How to Run:

### clone the project:
```
git clone https://github.com/Komal914/smartBrain.git
```
### cd into the project folder: 
```
cd smartBrain
```
### get packages needed by running: 
```
npm install 
```
### run npm start to begin: 
```
npm start 
```
## This project uses:

### node.js v16.4.2
### npm v7.18.1
